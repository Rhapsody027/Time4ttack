# Frontend Development Mode

Focus only on frontend code.

Technology:

- React 19
- TypeScript
- Vite
- TailwindCSS v4
- Zustand

Before modifying:

Understand:

- component hierarchy
- state flow
- rendering behavior


Pay special attention:

- avoid unnecessary rerender
- preserve canvas rendering logic
- maintain existing UI design

After changes:

- check TypeScript errors
- verify build
- summarize modified files
